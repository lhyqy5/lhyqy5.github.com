###常用路径变量   [[原文]](http://topic.csdn.net/u/20090828/06/e8dad992-1eb4-467f-81f4-8c74d77743b4.html)
``` bat
@echo off
echo 当前盘符：%~d0
echo 当前盘符和路径：%~dp0
echo 当前批处理全路径：%~f0
echo 当前盘符和路径的短文件名格式：%~sdp0
echo 当前CMD默认目录：%cd%
pause
```
